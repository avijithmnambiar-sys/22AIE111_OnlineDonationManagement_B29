## Problem Description

The Online Donation Management System is a console-based Java application that allows donors to make donations to charitable organizations. The system records donations, processes payments, tracks organization funds, and generates reports.

### Features
- Donor registration and password-based login
- Make donations to organizations
- Organization fund tracking
- Admin login and report generation
- Menu-driven interface

This system improves transparency and accountability in donation management.

## How to Run the Code

1. Open the project in Eclipse, IntelliJ IDEA, or VS Code.
2. Ensure all Java files are inside the package:

   `javaProject`

3. Compile all `.java` files.
4. Run `Main.java`.
5. Follow the menu displayed in the console.

### Files Required
- User.java
- Donor.java
- Admin.java
- Organisation.java
- Donation.java
- Payment.java
- Report.java
- Main.java
